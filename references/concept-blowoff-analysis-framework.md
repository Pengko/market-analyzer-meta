# 概念炒作退潮分析框架

> 当用户问"为什么这只股票跌停/暴跌"时，使用此框架进行诊断分析。
> 适用于：短期暴涨后暴跌、板块集体退潮、概念炒作崩塌等场景。

## 适用场景

- 用户问"为什么今天跌停"而非"明天怎么看"
- 需要诊断原因而非预测走势
- 概念驱动型暴涨后的回调分析

## A股概念炒作标准生命周期

```
催化期 (1-2天)
  ↓ 利好消息/政策/合作公告发布
  ↓ 少数先知先觉资金介入
发酵期 (1-3天)
  ↓ 消息扩散，媒体跟进
  ↓ 涨停板/连板出现
  ↓ 散户开始注意到
加速期 (1-3天)
  ↓ 板块集体涨停
  ↓ 舆论高峰（"估值重塑""XX时代"等叙事）
  ↓ 散户蜂拥而入，成交量暴增
分歧期 (1-2天)
  ↓ 高位放量滞涨（量大但不涨）
  ↓ 聪明钱开始撤退
  ↓ 资金流向显示大单流出
崩塌期 (1-3天)
  ↓ 利好出尽 / 概念证伪 / 利空催化
  ↓ 跌停或暴跌
  ↓ 踩踏出逃
```

**典型总周期**：5-10个交易日（从催化到崩塌）
**关键特征**：概念炒作赚的是情绪的钱，不是业绩的钱；情绪来得快去得也快

## 主力盈利分析方法（大盘子也能赚钱）

### 误区

"这只股票2000亿市值，主力拉不动/没利润" —— **错误**

### 正确逻辑

主力不需要控制全部市值，只需要控制流通盘的3-5%即可获利：

| 步骤 | 操作 | 关键指标 |
|------|------|----------|
| 积累期 | 低位悄悄买入3-5%流通盘 | 成交量小、股价波动低 |
| 拉升期 | 配合利好拉涨停 | 封板资金相对小（流通股持有者惜售） |
| 出货期 | 高位卖给追涨散户 | 成交量暴增、资金流向显示净流出 |

### 主力利润估算公式

```
假设持仓 = 流通盘 × 3-5%
买入成本 = 持仓股数 × 积累期均价
卖出收入 = 持仓股数 × 出货期均价
利润 = 卖出收入 - 买入成本
回报率 = (出货均价 / 积累均价 - 1) × 100%
```

### 实战案例（京东方A 2026-05-21至05-29）

| 维度 | 数据 |
|------|------|
| 流通盘 | ~363亿股 |
| 积累期（5/13-20） | 日均成交~26亿元，均价~4.2元 |
| 出货期（5/21-28） | 日均成交~221亿元，均价~5.5元 |
| 6天总成交 | ~1327亿元（流通市值的70%） |
| 主力假设持仓3% | 10.9亿股 |
| 买入成本 | ~46亿元 |
| 卖出收入 | ~60亿元 |
| **利润** | **~14亿元（+30%回报）** |

### 关键观测指标

判断主力是否在出货：
1. **涨停日资金净流出**：涨停但整体资金净流出 = 主力在涨停板上出货（如5/21净流出48.6亿）
2. **高位放量滞涨**：成交量大但价格不涨 = 买卖双方都在大量换手
3. **大单vs小单分化**：大单卖出+小单买入 = 机构出散户接
4. **量能衰减**：从天量逐步缩量 = 最后的追涨资金消耗殆尽

## 诊断分析输出模板

当分析"为什么暴跌"时，输出结构：

1. **直接原因**：板块退潮 / 利空催化 / 情绪崩塌 / 获利回吐
2. **消息面时间线**：催化→发酵→加速→分歧→崩塌，标注每个阶段的新闻和股价
3. **成交量分析**：积累期 vs 出货期的成交额对比
4. **资金流向证据**：哪天开始大单流出，净流出多少
5. **主力盈亏估算**：积累期均价、出货期均价、理论利润
6. **回答用户核心疑问**：为什么热度这么快？（答：不是快，是标准周期）大盘子能赚钱吗？（答：只需控制小比例即可）

## 常见用户追问及标准回答

**"为什么热度这么快就过了？"**
→ 不是热度"过了"，而是概念炒作的标准生命周期就是5-10天。热度来得快去得也快，因为赚的是情绪的钱，不是业绩的钱。

**"大盘子主力有盈利吗？"**
→ 绝对有。主力不需要动2000亿，只需控制3-5%流通盘。通过低位积累+高位出货，一周可赚20-30%回报。涨停日的净流出数据就是铁证。

**"有什么消息吗？"**
→ 很多时候没有新消息。概念退潮不需要利空催化——利好出尽本身就是最大的利空。当所有想买的人都已经买了，剩下的只有卖方。
