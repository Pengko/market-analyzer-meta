# 消息面获取渠道说明

## 目标

给股票深度分析 skill 提供统一的消息获取顺序，避免分析时：

- 只看单一平台
- 只抄题材标签
- 把传闻当公告
- 找不到 API 就直接跳过消息面

## 默认执行顺序

1. `公告 / 交易所`
   - 巨潮资讯
   - 上交所 / 深交所
   - 公司公告
2. `政策 / 官方`
   - 证监会
   - 工信部
   - 发改委
   - 政府官网
3. `主流财经媒体`
   - 财联社
   - 证券时报
   - 上证报
   - 中证报
4. `行情平台 / 题材归因`
   - 东方财富
   - 同花顺
   - 开盘啦
   - Yahoo Finance Hong Kong
5. `社区 / 二手传播`
   - 股吧
   - 社媒
   - 群转发

## 每层用途

- `公告 / 交易所`：确认事实
- `政策 / 官方`：确认监管或行业催化
- `主流财经媒体`：确认公开传播
- `行情平台 / 题材归因`：确认市场如何解释
- `社区 / 二手传播`：观察情绪噪音

## 默认规则

- 无 API 时，默认走浏览器
- 低可信渠道不能单独主导结论
- 行情平台可以解释题材热度，但不能替代公告事实
- Yahoo Finance Hong Kong 适合作为补充传播渠道，不适合作为 A 股个股催化的一手实锤

## 最低合格记录

分析输出里至少记录：

- 查了哪些渠道
- 每层有没有拿到有效信息
- 最终采用了哪条消息作为主要驱动
- 这条消息的可信度和级别是什么
