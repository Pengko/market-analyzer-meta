# 大盘板块分析

从宏观到微观，先看大盘环境，再找板块热点，最后落到具体个股。

## 分析顺序

```
大盘环境 → 板块热点 → 龙头个股 → 消息催化 → 交易结论
```

与个股分析（stock-deep-analysis）的区别：
- 个股分析：从一只股票出发，扩散到板块和大盘
- 大盘板块分析：从大盘出发，收缩到板块和个股

## Step 1: 大盘环境分析

**必须先完成，不可跳过。**

### 数据获取
- 上证指数：`qt.gtimg.cn/q=sh000001`
- 深证成指：`qt.gtimg.cn/q=sz399001`
- 创业板指：`qt.gtimg.cn/q=sz399006`

### 分析维度
1. **指数强弱**：三大指数涨跌幅、是否共振
2. **成交量**：两市成交额 vs 近期均量（放量/缩量）
3. **风格偏向**：大盘股 vs 小盘股、价值 vs 成长
4. **风险偏好**：涨停家数、跌停家数、炸板率
5. **北向资金**：净流入/流出方向

### 输出格式
```
## 一、大盘环境

| 指数 | 收盘 | 涨跌幅 | 成交额 | 量能 |
|------|------|--------|--------|------|
| 上证 | XXXX | +X.XX% | XXX亿 | 放量/缩量 |

- 市场整体：偏强/中性/偏弱
- 风险偏好：高/中/低
- 对目标分析：顺风/逆风/中性
```

## Step 2: 板块热点分析

### 数据获取
- KPL 概念数据：`theme_data/kpl_concept_cons/`
- DC 概念数据：`theme_data/dc_concept/`
- 行业资金流：`moneyflow_data/`

### 分析维度
1. **当日热点**：涨幅前 5 板块、资金流入前 5 板块
2. **持续性**：近 3 日连续走强的板块
3. **轮动阶段**：加强/分化/轮动/退潮
4. **龙头识别**：每个热点板块的龙头股
5. **资金结构**：大单/中单/小单流向

### 输出格式
```
## 二、板块热点

| 板块 | 涨幅 | 资金流向 | 龙头股 | 阶段 |
|------|------|----------|--------|------|
| AI硬件 | +3.5% | 净流入 | 永鼎股份 | 加强 |

- 当前主线：XX板块
- 潜在接力：XX板块
- 退潮风险：XX板块
```

## Step 3: 龙头个股分析

从 Step 2 识别的热点板块中，选出 3-5 只重点个股：

### 选股标准
1. 板块龙头（涨幅最大、封板最早）
2. 板块中军（成交额最大、机构参与）
3. 高弹性标的（振幅最大、换手最高）

### 分析内容
- 日线趋势
- 筹码结构（cyq_perf）
- 资金流向
- 龙虎榜（如有）

### 输出格式
```
## 三、重点个股

| 股票 | 角色 | 收盘 | 涨跌幅 | 核心逻辑 |
|------|------|------|--------|----------|
| XX股份 | 龙头 | XX.XX | +X.XX% | 涨停封板，资金集中 |
```

## Step 4: 消息催化分析

### 数据获取
- TrendRadar MCP：`get_latest_news` + `get_latest_rss`
- 本地新闻：`news_pipeline/`

### 分析维度
1. **宏观政策**：央行、发改委、证监会等政策动向
2. **行业催化**：涨价、供需变化、技术突破
3. **市场事件**：IPO、回购、并购重组

### 输出格式
```
## 四、消息催化

| 消息 | 影响板块 | 影响程度 | 方向 |
|------|----------|----------|------|
| XX政策 | AI硬件 | 板块级 | 偏利多 |
```

## Step 5: 交易结论

综合以上分析，给出：
1. **市场判断**：偏多/中性/偏空
2. **板块建议**：重点关注哪些板块、回避哪些板块
3. **个股建议**：可以关注的个股和介入条件
4. **风险提示**：需要警惕的风险点

### 输出格式
```
## 五、交易结论

- 市场判断：偏强，可积极参与
- 重点板块：AI硬件（加强阶段）、面板（轮动中）
- 关注个股：XX股份（等回踩确认）、XX科技（突破后介入）
- 风险位：上证 XXX 点（跌破则减仓）
```

## LLM Prompt 侧重

与个股分析不同，大盘板块分析的 LLM prompt 应侧重：
- 宏观环境对板块的传导逻辑
- 板块轮动规律和持续性判断
- 龙头股的辨识和跟风股的区分
- 市场情绪周期（冰点→回暖→高潮→退潮）

## 数据源

复用 `stock-deep-analysis` 的数据层：
- 大盘指数：`index_daily/` + 腾讯 API
- 板块数据：`theme_data/` (KPL + DC)
- 资金流向：`moneyflow_data/`
- 消息面：TrendRadar MCP + 本地 news_pipeline
