# 消息面驱动分析

从消息出发，映射到板块，再落到个股，最后结合大盘环境判断。

## 分析顺序

```
消息获取 → 消息解读 → 板块映射 → 受益/受损个股 → 大盘环境 → 交易结论
```

与个股分析的区别：
- 个股分析：从一只股票出发，消息面是补充维度
- 消息面分析：从消息出发，个股是最终落脚点

## Step 1: 消息获取

### 数据源优先级
1. **TrendRadar MCP**（主数据源）
   - `get_latest_news(limit=500)` — 全量热榜
   - `get_latest_rss(limit=500, days=3)` — 全量 RSS
2. **本地 news_pipeline**（降级）
   - `~/quant-data/tushare/消息面数据/raw/news_pipeline/YYYY/MM/DD/`
3. **浏览器抓取**（兜底）
   - 东方财富、财联社、新浪财经

### 消息分类
| 类型 | 来源 | 影响范围 |
|------|------|----------|
| 宏观政策 | 央行/发改委/国务院 | 全市场/特定行业 |
| 行业新闻 | 产业动态/涨价/供需 | 特定板块 |
| 公司公告 | 巨潮/交易所 | 个股 |
| 市场事件 | IPO/回购/并购 | 个股/板块 |
| 突发事件 | 地缘/自然灾害 | 全市场 |

### 输出格式
```
## 一、消息概览

| 时间 | 消息 | 来源 | 级别 | 新鲜度 |
|------|------|------|------|--------|
| 10:30 | XX政策出台 | 新华社 | 市场级 | 盘中新增 |
| 09:00 | XX公司公告 | 巨潮 | 个股级 | 当日新增 |

- 消息总量：XX条
- 有效消息：XX条
- 噪音过滤：XX条
```

## Step 2: 消息解读

对每条有效消息进行深度解读：

### 解读维度
1. **消息本质**：实际影响 vs 市场预期
2. **传导路径**：消息→哪些行业→哪些公司
3. **时效性**：短期刺激 vs 长期逻辑
4. **市场定价**：是否已被 price in
5. **情感方向**：偏利多/偏利空/中性/多空交织

### 输出格式
```
## 二、消息解读

### 消息1：XX政策出台
- 本质：XX
- 传导：政策 → XX行业 → XX公司
- 时效：短期刺激（1-3天）
- 定价：尚未充分反映
- 方向：偏利多
```

## Step 3: 板块映射

将消息映射到具体板块：

### 映射方法
1. **直接映射**：消息直接提及的行业/公司
2. **产业链映射**：上游/中游/下游传导
3. **概念映射**：TrendRadar 概念标签匹配
4. **历史类比**：类似消息历史上的板块反应

### 输出格式
```
## 三、板块映射

| 消息 | 受益板块 | 受损板块 | 映射逻辑 |
|------|----------|----------|----------|
| XX政策 | AI硬件、半导体 | 传统制造 | 政策扶持方向 |

- 核心受益板块：XX（直接受益）
- 间接受益板块：XX（产业链传导）
- 潜在受损板块：XX（替代/竞争）
```

## Step 4: 受益/受损个股分析

从板块映射中，选出具体个股：

### 选股标准
1. **直接受益**：公告/消息直接提及的公司
2. **板块龙头**：受益板块的龙头股
3. **高弹性**：历史上对消息反应最敏感的标的
4. **低位优先**：尚未大涨、有空间的标的

### 分析内容（复用 stock-deep-analysis 的分析模块）
- 日线趋势
- 筹码结构
- 资金流向
- 龙虎榜（如有）

### 输出格式
```
## 四、受益个股

| 股票 | 板块 | 受益逻辑 | 当前位置 | 建议 |
|------|------|----------|----------|------|
| XX股份 | AI硬件 | 政策直接受益 | 低位 | 关注突破 |
| XX科技 | 半导体 | 产业链传导 | 中位 | 等回踩 |
```

## Step 5: 大盘环境验证

最后验证大盘环境是否支持消息驱动的交易机会：

### 验证维度
1. **大盘趋势**：消息驱动的个股在大盘顺风时更容易兑现
2. **板块共振**：受益板块是否已有资金关注
3. **市场情绪**：当前是高潮期还是冰点期
4. **资金结构**：是否有增量资金进场

### 输出格式
```
## 五、大盘环境验证

- 大盘趋势：偏强，顺风环境
- 板块共振：AI硬件已有资金流入，消息催化有望加速
- 市场情绪：回暖期，适合积极参与
- 资金结构：北向净流入，增量资金进场
```

## Step 6: 交易结论

综合以上分析，给出：

1. **消息评级**：强催化/中等催化/弱催化/噪音
2. **交易方向**：做多/观望/回避
3. **重点标的**：最值得关注的 2-3 只股票
4. **介入条件**：什么价位/什么信号可以介入
5. **风险提示**：消息证伪风险、利好兑现风险

### 输出格式
```
## 六、交易结论

- 消息评级：强催化（政策级别，影响范围广）
- 交易方向：做多
- 重点标的：
  1. XX股份 — 政策直接受益，低位突破
  2. XX科技 — 产业链传导，等回踩确认
- 介入条件：XX股份突破XX元可介入
- 风险位：跌破XX元止损
- 失效条件：政策细则不及预期
```

## LLM Prompt 侧重

消息面分析的 LLM prompt 应侧重：
- 消息的真实影响 vs 市场炒作
- 产业链传导逻辑
- 历史类比和概率推演
- 利好/利空的定价程度
- 消息时效性判断（短期刺激 vs 长期逻辑）

## 数据源

复用 `stock-deep-analysis` 的数据层：
- 消息面：TrendRadar MCP + 本地 news_pipeline
- 板块数据：`theme_data/` (KPL + DC)
- 个股数据：`daily/` + `cyq_perf/` + `moneyflow_data/`
- 龙虎榜：`top_list/`
