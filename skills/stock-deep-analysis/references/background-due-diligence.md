# 背调框架 (Background Due Diligence)

> 2026-06-12 新增：基本面分析必须包含风险背调，不能只看业务逻辑不查风险。

## 一、触发条件

当基本面分析涉及以下场景时，**必须**附加背调：
- 用户问"这家公司有没有风险"
- 深度分析某只股票的基本面
- 用户准备建仓前的尽职调查
- 涉及重组/定增/收购等重大事项

## 二、背调维度（8大类）

### 1. 诉讼与仲裁

**数据源**：
- 中国裁判文书网：`https://wenshu.court.gov.cn/`
- 天眼查/企查查：诉讼信息模块
- 巨潮资讯：公司公告中的诉讼披露

**关注点**：
- 未决诉讼金额占净资产比例（>10% 为高风险）
- 是否涉及核心资产的权属纠纷
- 是否有集体诉讼/投资者索赔
- 诉讼对方是否为公司实控人/关联方

### 2. 监管处罚

**数据源**：
- 证监会处罚决定：`http://www.csrc.gov.cn/csrc/c100028/common_list.shtml`
- 交易所监管函/问询函/谴责
- 同花顺/东方财富个股F10"公司公告"中的处罚记录

**关注点**：
- 近3年是否有行政处罚
- 是否被ST/*ST过
- 问询函是否涉及财务造假嫌疑
- 是否有立案调查（最严重）

### 3. 大股东/实控人风险

**数据源**：
- 巨潮资讯：股东变动公告
- 交易所互动平台
- 东方财富F10"股东研究"

**关注点**：
- 大股东质押比例（>70% 高风险，>50% 关注）
- 大股东减持计划/历史减持
- 实控人是否被限制消费/冻结资产
- 是否存在一致行动人变更/控制权争夺

### 4. 关联交易

**数据源**：
- 年报中的关联交易披露
- 巨潮资讯公告

**关注点**：
- 关联交易金额占营收比例（>30% 高关注）
- 关联交易定价是否公允
- 是否存在通过关联交易虚增收入的嫌疑
- 关联方应收账款/预付账款异常

### 5. 财务异常信号

**数据源**：
- 审计报告（标准无保留 vs 带强调事项 vs 保留意见 vs 否定意见）
- 财务报表附注

**关注点**：
- 审计意见类型（非标=重大风险信号）
- 经营现金流长期低于净利润（可能虚增利润）
- 应收账款增速远超营收增速
- 存货异常膨胀
- 商誉占净资产比例>30%（减值风险）
- 在建工程长期不转固（可能藏费用）
- 其他应收款异常（"垃圾桶"科目）

### 6. 信息披露违规

**数据源**：
- 交易所监管措施/处分
- 证监会行政处罚

**关注点**：
- 是否有信息披露违规记录
- 是否存在业绩预告大幅修正
- 是否有重大事项未及时披露
- 互动平台回复是否回避核心问题

### 7. 环保/安全/合规

**数据源**：
- 生态环境部处罚公告
- 应急管理部门事故通报
- 行业监管动态

**关注点**：
- 近3年是否有环保处罚
- 是否发生过安全生产事故
- 是否属于高污染/高能耗行业且面临政策收紧
- 产品是否涉及进出口管制/制裁

### 8. 舆情与口碑

**数据源**：
- 股吧/雪球讨论
- 新闻搜索
- 消费者投诉平台（黑猫投诉等）

**关注点**：
- 是否有大规模负面舆情
- 产品/服务质量口碑
- 是否有"暴雷"传闻（需交叉验证）
- 社交媒体上的员工评价

## 三、背调输出格式

```
### 风险背调

| 维度 | 状态 | 风险等级 | 说明 |
|------|------|----------|------|
| 诉讼仲裁 | ✅无重大诉讼 / ⚠️有未决诉讼 / ❌重大纠纷 | 低/中/高 | 具体说明 |
| 监管处罚 | ✅无处罚 / ⚠️有问询 / ❌有处罚 | 低/中/高 | 具体说明 |
| 大股东质押 | ✅低质押 / ⚠️中等质押 / ❌高质押 | 低/中/高 | 质押比例XX% |
| 关联交易 | ✅正常 / ⚠️偏高 / ❌异常 | 低/中/高 | 占比XX% |
| 财务质量 | ✅标准无保留 / ⚠️有强调事项 / ❌非标意见 | 低/中/高 | 审计意见类型 |
| 信息披露 | ✅规范 / ⚠️有瑕疵 / ❌有违规 | 低/中/高 | 具体说明 |
| 环保安全 | ✅合规 / ⚠️有整改 / ❌有处罚 | 低/中/高 | 具体说明 |
| 舆情口碑 | ✅正面 / ⚠️中性 / ❌负面 | 低/中/高 | 具体说明 |

**综合风险评级**: 低风险 / 中等风险（需关注） / 高风险（建议回避）
**核心风险点**: 列出最需要警惕的1-3个风险
**背调结论**: 是否影响建仓决策
```

## 四、快速背调流程（5分钟版）

当时间有限时，按优先级快速排查：

1. **审计意见**（1分钟）：查最新年报审计意见类型 → 非标=直接标红
2. **大股东质押**（1分钟）：东方财富F10股东研究 → 质押比例>50%标黄，>70%标红
3. **诉讼处罚**（2分钟）：天眼查/企查查 → 近3年行政处罚+重大诉讼
4. **商誉减值**（1分钟）：年报商誉/净资产 → >30%标黄

## 五、常见陷阱

1. **只看业务不看风险**：业务逻辑再好，财务造假/监管处罚可以直接归零
2. **忽略审计意见**：非标审计意见是最强的风险信号，但容易被忽略
3. **质押比例陷阱**：高质押≠一定出问题，但高质押+股价下跌=强平风险
4. **关联交易隐蔽**：年报关联交易章节藏在附注里，不主动查很难发现
5. **诉讼时效**：部分诉讼可能已经结案但影响仍在（如赔偿执行）
