# 概念板块归属验证方法

> 创建于 2026-06-06，内容可能随技术迭代和市场变化过时，使用时请注意时效性。

## 适用场景

用户问"某只股票有没有XX概念"、"XX概念成分股有哪些"、"A是否有B业务"时，需要验证个股与概念板块的实际关联。

## 验证层级

### 第一层：概念板块成分股查证（最可靠）

**同花顺F10浏览器法**（最可靠，推荐首选）：
1. `browser_navigate` → `https://stockpage.10jqka.com.cn/{code}/`
2. 点击 F10 → 概念题材 → 查看完整概念列表
3. 概念列表包含：概念名称 + 概念解析（通常是互动平台/公告原文摘录）
4. **判断标准**：目标概念是否出现在列表中

**东方财富搜索API**：
```
curl -s "https://searchapi.eastmoney.com/api/suggest/get?input={概念名}&type=14&token=D43BF722C8E33BDC906FB84D85E326E8&count=10"
```
返回板块代码（如 BK1175=玻璃基板），再查成分股。**注意**：push2.eastmoney.com 的成分股API在周末或非交易时段可能返回空响应。

### 第二层：业务实质验证（防止误判）

即使个股出现在某概念成分股列表中，仍需验证业务实质：
- 公司主营业务是否真正覆盖该概念的核心技术/产品
- 收入占比是否足够大（如<2%则概念纯度极低）
- 是否只是间接关联（供应链相邻、客户重叠等）

### 第三层：情绪关联 vs 业务关联

区分两种"沾边"方式：
- **业务关联**：公司实际有该概念对应的产品/技术/收入
- **情绪关联**：公司因供应链/行业属性被市场情绪带动，但无实际业务

示例：TCL科技在玻璃基板概念上涨期间跟涨，原因是"面板产业链整体预期提升"的情绪带动，但公司实际业务是LCD/OLED显示面板，不做芯片封装用玻璃基板。

## 常见概念边界混淆点

同名或近名概念可能指完全不同的技术/产品。验证时要区分清楚：

| 混淆类型 | 说明 |
|---------|------|
| 同名不同技术 | 如"玻璃基板"：显示面板用的 vs 芯片封装用的，技术路线和产业链完全不同 |
| 上下游混淆 | 如"OLED面板" vs "OLED材料"，做面板的和做材料的是不同产业链位置 |
| 大类混淆 | 如"半导体材料" vs "半导体设备" vs "半导体设计"，都带"半导体"但赛道不同 |
| 成熟度混淆 | 如"MiniLED"（成熟量产）vs "MicroLED"（仍在研发），同一家公司可能都布局但阶段不同 |

**验证方法：**
1. 查该概念的核心技术/产品是什么
2. 查目标公司实际做什么
3. 对比两者是否匹配，而非只看名字相似

## 验证流程模板

```
用户问：XX科技有YY业务吗？

1. 同花顺F10查概念列表 → 是否有"YY概念"
2. 若有 → 查概念解析中的业务描述 → 是否为实质性业务
3. 若无 → 查主营业务是否间接涉及 → 判断是否为情绪关联
4. 查该概念股近期行情驱动原因 → 区分业务驱动 vs 情绪带动
5. 给出明确结论：有/无/有但占比低/间接关联
```

## 合作公告/MOU 评估方法

当公司发布合作备忘录(MOU)、战略合作协议等公告时，需要区分公告的实质与市场解读的落差。

### MOU vs 正式合同

| 类型 | 法律约束力 | 市场影响特征 | 解读重点 |
|------|-----------|-------------|---------|
| 战略合作**备忘录** (MOU) | **无**法律约束力 | 短期情绪催化强，中期可随时终止 | 关注是否后续转为正式合同 |
| 框架**协议** | 部分约束（视条款） | 中期催化，看合作范围和期限 | 关注约束条款和排他性 |
| 正式**合同/订单** | **有**法律约束力 | 强催化，可直接测算业绩影响 | 关注金额、期限、交付节奏 |

### 评估清单

1. **合作深度**：MOU → 框架协议 → 正式合同？（阶段越早，不确定性越大）
2. **业绩贡献时间线**：公司自述"未来X年无法产生重大影响"？→ 远期期权
3. **技术验证状态**：试验线/中试线 → 量产 → 批量供货？（阶段越早，风险越大）
4. **与终端需求方关系**：是与中间商还是终端客户直接合作？
5. **公司自身风险提示**：上市公司公告中的风险提示是最可靠的信息源，必须重视

### 常见叙事陷阱

- **产业链臆想**：市场自行搭建"A→B→C"产业链叙事，但公司与终端需求方可能并无直接关系
- **概念纯度**：公司原有业务与新方向之间的技术迁移难度被低估（如显示面板→芯片封装，虽都涉及玻璃但工艺差距巨大）
- **重投入约束**：新业务需要重资产投入，但公司自身盈利能力可能不足以支撑

> 详细案例见 `references/glass-substrate-packaging-technology.md`（京东方A × 康宁 MOU 案例分析）

## 已知坑

- **东方财富push2 API周末空响应**：周末和非交易时段，`push2.eastmoney.com` 的成分股API可能返回空，不是接口挂了
- **akshare连接中断**：akshare底层依赖的东财接口在高峰期可能被限流，需要fallback到浏览器
- **搜索引擎验证码**：百度/Google/Bing在无代理环境下可能触发验证码，优先用TrendRadar RSS搜索或直接浏览器访问同花顺/东方财富
- **概念动态≠实际业务**：同花顺"概念动态"显示某股"新增XX概念"只是平台归类变化，不代表公司新增了该业务
