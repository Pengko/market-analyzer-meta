# 跨板块相关性分析方法

> 适用场景：个股属于多个概念板块（如三安光电既是LED龙头，又有先进封装/半导体属性），需量化判断其当日走势实际跟随哪个板块。

## 分析流程

### 1. 确定候选板块

从个股概念标签中提取2-3个最相关的板块，每个板块选3-5只代表股：
- **龙头**（板块内市值最大或辨识度最高）
- **中军**（跟随性强、成交额大）
- **先手**（最先启动或涨幅最大）
- **独立逻辑**（与目标股有可比逻辑但不完全相同）

### 2. 获取分时数据

用新浪5分钟K线API批量获取：
```
https://money.finance.sina.com.cn/quotes_service/api/json_v2.php/CN_MarketData.getKLineData?symbol={code}&scale=5&ma=no&datalen=60
```

提取今日数据，计算各关键时间点相对昨收的涨跌幅。

### 3. 两层相关性计算

**第一层：日线相关性（近5日收盘涨跌）**
```python
# 用numpy计算Pearson相关系数
corr = np.corrcoef(target_daily_changes, peer_daily_changes)[0,1]
```
判断中期趋势是否同步。

**第二层：分时相关性（当日关键时窗涨跌幅）**
```python
# 关键时窗：09:35, 09:45, 09:50, 10:05, 13:15, 14:15, 14:55
corr = np.corrcoef(target_intraday_changes, peer_intraday_changes)[0,1]
```
判断当日盘中节奏是否同步。

### 4. 判断标准

| 相关系数 | 日线层面 | 分时层面 |
|---------|---------|---------|
| r > 0.8 | 高度一致 | 盘中节奏几乎同步 |
| 0.6 < r < 0.8 | 中等相似 | 大方向一致但幅度不同 |
| r < 0.6 | 差异较大 | 独立走势 |

### 5. 输出结论模板

```
板块相关性分析:
  [板块A名称] 日线 r=X.XX / 分时 r=X.XX → 诊断
  [板块B名称] 日线 r=X.XX / 分时 r=X.XX → 诊断
  综合判断: 该股今日走势更接近[板块X]，[原因]
```

## 常见多属性板块

| 个股 | 主板块 | 副板块 | 分析要点 |
|------|--------|--------|---------|
| 三安光电 | LED芯片 | 先进封装/SiC/MiniLED | LED vs 封装的分时相关性 |
| 东山精密 | 消费电子 | FPC/先进封装/苹果链 | 消费电子 vs 封装 |
| 通富微电 | 封装 | AMD产业链 | 封装 vs AMD概念 |
| 华天科技 | 封装 | 军工/存储 | 封装 vs 军工 |

## 注意事项

- 分时相关性反映的是"今天谁带它"，不等于长期归属
- 单日分时相关性高可能是巧合，需结合日线层面确认
- 如果日线和分时都指向同一板块，可以较确信地判断跟随关系
- 如果日线和分时指向不同板块，说明该股处于"多空拉锯"状态
