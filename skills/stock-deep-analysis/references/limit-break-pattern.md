# 涨停炸板分时分析判断框架

> 适用场景：个股当日冲击涨停价或触及涨停价，但未能封板，随后回落收涨。
> 本文档为 stock-deep-analysis 技能的补充参考，用于规范化该场景下的分析流程与关键判断指标。

---

## 一、核心判断指标：冲板量能 vs 回落量能

当个股在盘中冲击涨停价但未能封板时，最关键的量价对比是：

| 比较维度 | 冲板时段 | 炸板回落时段 | 判断逻辑 |
|-----------|-----------|-------------|----------|
| 量能 | 冲板累计成交量 | 炸板后回落累计成交量 | 冲板量 < 回落量 → 板上抛压大于买盘，封板意愿不足 |
| 成交额 | 冲板累计成交额 | 炸板后回落累计成交额 | 回落金额远超冲板 → 主力在板上放量出货 |
| 均价位置 | 冲板均价 vs 封板价 | 回落均价 vs 封板价 | 均价远离封板价 → 放量区间低于封板价，出货主导 |

### 量能比判断规则

- **冲板量能 ≥ 2× 回落量能**：强买盘主导，炸板是意外（被大单撞开或市场回落带下），主力本意是封板
- **冲板量能 ≈ 回落量能**：强强分歧，板上换手充分，第二天走势不确定性高
- **冲板量能 < 回落量能**：弱封板，板上主力出货主导，回落可能继续
- **冲板量能 ≤ 1/2 × 回落量能**：异常弱封板，主力封板意愿极弱，放量出货明显，优先判定为"分发"或"主力出货"

---

## 二、分钟线分析框架

### 时段划分

分析冲板炸板时，需要明确划分以下时段：

1. **冲板前蓄势阶段**：从当天开盘到第一次冲击涨停前的价量走势
2. **冲板时段**：股价触及涨停价的时段，通常持续 1-5 分钟
3. **炸板回落时段**：从涨停价被撞开到回落稳定的时段，通常是冲板后 5-15 分钟
4. **回落稳定后阶段**：从回落稳定到盘中其他重要时点（如午后开盘、尾盘）

### 量能日志记录模板

```
| 时段 | 价格区间 | 累计成交量（手） | 累计成交额（元） | VWAP（元） | 主力行为 |
|------|----------|-------------------|------------------|----------|----------|
| 冲板前蓄势 | X.XX~X.XX | XX.X万 | XXXX万 | X.XX | 描述 |
| 冲板时段 | X.XX~涨停价 | XX.X万 | XXXX万 | X.XX | 描述 |
| 炸板回落 | 涨停价~X.XX | XX.X万 | XXXX万 | X.XX | 描述 |
| 回落稳定后 | X.XX~X.XX | XX.X万 | XXXX万 | X.XX | 描述 |
```

**重要**：VWAP 计算时，`volume` 字段单位为**手**（即 100 股），必须先乘以 100 转换为股数后再计算：
```
vwap = cum_amount / (cum_volume * 100)
```

## 三、补充判断指标（2026-06-12 黄河旋风实战新增）

### 1. 涨停时段量占全天比

除"冲板量 vs 回落量"外，增加一个更直观的指标：

```
涨停时段量占比 = 涨停相关时段成交量 / 全天总成交量

判断标准:
  > 40% → 极高，板上换手剧烈，抛压极重
  20%~40% → 偏高，板上分歧明显
  < 20% → 正常，炸板更多是被动带开
```

**案例: 黄河旋风(600172) 2026-06-12**
- 涨停时段(09:40-09:45)成交: 8954万手
- 全天总成交: 20892万手
- 涨停量占比: **42.9%** → 极高，板上抛压极重

### 2. 多次触板分析

当一个交易日内多次冲击涨停均失败时（如黄河旋风09:40和09:45两次触板）：

| 触板次数 | 判断 |
|----------|------|
| 1次触板炸开 | 可能是被动炸板（大盘拖累/大单撞击），需看回落量 |
| 2次触板均炸 | 主力封板意愿不足，或套牢盘抛压极重 |
| ≥3次触板均炸 | 主力大概率在利用涨停吸引跟风盘出货 |

**多次触板的量价特征**：
- 第二次触板量 > 第一次 → 抛压在加速释放，更危险
- 第二次触板量 < 第一次 → 买盘在衰竭，后续更难封住
- 每次触板后回落低点不断下移 → 多头信心在崩溃

### 3. 内外盘比作为炸板确认信号

```
内盘(主动卖) > 外盘(主动买) → 板上主动卖出占优，抛压偏重
内盘/外盘比 > 55% → 强烈偏空信号
```

**案例: 黄河旋风**
- 外盘(主动买): 46.3%
- 内盘(主动卖): 53.7%
- 结论: 主动卖出更多，与涨停炸板一致

### 4. 涨停位置与缺口/均线的交互

涨停价恰好处于技术压力位时，炸板概率显著上升：

| 涨停价位置 | 炸板概率 | 说明 |
|-----------|---------|------|
| 涨停价上方有未回补向下缺口 | 高 | 缺口+涨停双重压力 |
| 涨停价接近MA20/MA30 | 中高 | 均线压力叠加 |
| 涨停价突破前高 | 中 | 解套盘涌出 |
| 涨停价处于无压力区 | 低 | 仅获利盘，抛压较轻 |

**案例: 黄河旋风**
- 涨停价13.00，上方未回补向下缺口13.17~13.26
- 涨停价恰好被缺口压制 → 炸板

## 四、配套判断维度（板块/市场/前期）

### 1. 板块联动检查

冲板炸板时，必须检查同板块龙头是否同步出现类似走势：
- 若板块多只个股同时冲板炸板 → 板块分歧/退潮，不是个股独立问题
- 若只有目标股冲板炸板 → 主力意图问题，需进一步检查是否有个股特定利空

### 2. 市场环境检查

- 若大盘当日强势，个股却冲板炸板 → 相对走弱，可能是被主动出货
- 若大盘当日弱势，个股冲板后被带开 → 可能是被大盘拖累

### 3. 前期走势检查

- 若近几日已有明显涨幅，冲板炸板更可能是高位分发
- 若前期持续低位整理，冲板炸板可能是试盘，后续可能有机会再度冲击

## 五、案例参考：华电辽能 600396.SH @ 2026-05-26

### 股票基本信息
- 代码：600396.SH
- 名称：华电辽能
- 当日涨跌幅：+4.19%
- 当日最高涨幅：+9.75%（触及涨停，未封板）
- 当日收盘价：7.45 元
- 涨停价：7.85 元

### 分时量能记录

| 时段 | 价格区间 | 累计成交量（手） | 累计成交额（元） | VWAP（元） | 主力行为 |
|------|----------|-------------------|------------------|----------|----------|
| 14:25-14:30 冲板前 | 7.14~7.65 | 约 12.4万 | 约 9193万 | 7.41 | 边拉边洗，放量探路 |
| 14:30-14:35 冲板 | 7.65~7.85 | 约 7.9万 | 约 6130万 | 7.76 | 快速冲击涨停，但封单薄弱 |
| 14:35-14:50 炸板回落 | 7.85~7.45 | 约 36.5万 | 约 27300万 | 7.48 | 被大单撞开，放量回落，主力出货 |
| 14:50-15:00 稳定 | 7.45~7.45 | 约 3.1万 | 约 2310万 | 7.45 | 横盘整理，缓慢收涨 |

### 量能比较
- 冲板时段成交量：7.9 万手
- 炸板回落时段成交量：36.5 万手
- **比例：冲板量 / 回落量 = 0.22** — 远小于 0.5

### 判断结论
- 主力行为：**分发/主力出货**
- 核心依据：冲板量不到回落量的 1/4，板上抛压极大
- 交易建议：不建议追高，若已持有应考虑减仓
- 次日预期：高开低走或平开回落概率较大，需要防践止跌

## 六、常见误区

1. **误区：把"触涨停"等同于"强势"**
   - 纠正：触涨停后未能封住，本身就是弱势信号
2. **误区：只看到"最高涨幅+9.75%"就认为主力强**
   - 纠正：必须结合回落量能判断主力意图
3. **误区：回落后收涨就认为下午有机会**
   - 纠正：回落后的稳定通常是主力出货完毕后的横盘整理，不是再攻信号
4. **误区：VWAP 计算时忘记 volume 单位转换**
   - 纠正：分钟线 CSV 的 volume 是**手**，必须 `* 100`
5. **误区：多次触板=主力强势**
   - 纠正：多次触板但均炸板，说明主力封板意愿不足或抛压极重，是更明确的弱势信号
6. **误区：炸板后只看单日数据**
   - 纠正：必须结合近期走势（是否有连板/大涨后的回调）和缺口位置综合判断
