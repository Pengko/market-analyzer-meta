# 连板后断板分析框架

> 适用场景：个股连续涨停（2板或以上）后，次日低开或大幅回调，形成"断板"形态。
> 本文档为 stock-deep-analysis 技能的补充参考，用于规范化连板股断板后的分析流程与关键判断指标。
> 与 `limit-break-pattern.md`（单日炸板分析）互补：本框架关注多日连板后的首个回调日。

---

## 一、核心判断指标

### 1. 板块背离度（最关键信号）

| 指标 | 计算方法 | 判断标准 |
|------|----------|----------|
| 板块当日涨跌幅 | 同板块3-5只龙头股平均涨跌幅 | 区间+X% |
| 个股当日涨跌幅 | 断板日实际涨跌幅 | -Y% |
| 背离度 | 板块均值 - 个股涨跌幅 | >10% = 严重背离，极强利空信号 |

**解读规则：**
- 背离度 > 10%：板块暴涨个股暴跌 = 游资撤退确认，主力不跟板块走
- 背离度 5-10%：板块强个股弱 = 分歧加大，需谨慎
- 背离度 < 5%：板块与个股同步 = 正常回调，非信号

**为什么板块背离度是最关键指标：**
- 连板股通常靠游资接力推动
- 当板块整体强势时，新游资应该涌入接力
- 如果板块强而个股弱 = 没有新资金接力 = 老资金在出货

### 2. 量能结构

| 阶段 | 量能特征 | 判断 |
|------|----------|------|
| 连板日 | 涨停日成交 vs 5日均量 | 量比>2 = 放量涨停，量比<1.5 = 缩量涨停 |
| 断板日 | 断板日成交 vs 连板日成交 | 断板量>连板量 = 恐慌出逃 |
| 换手率 | 断板日换手率 | >15% = 充分换手，<8% = 未充分换手 |

**关键判断：**
- 缩量涨停 → 放量断板 = 主力边拉边出，高位派发
- 放量涨停 → 放量断板 = 多空充分博弈，后续方向取决于承接
- 缩量涨停 → 缩量断板 = 无资金关注，自然回落

### 3. 外盘/内盘比

| 比值 | 含义 | 操作建议 |
|------|------|----------|
| >1.2 | 买压大于卖压 | 可能有承接 |
| 0.9-1.2 | 多空平衡 | 观望 |
| <0.9 | 卖压大于买压 | 减仓/止损 |

### 4. 封单强度（连板日）

| 封单量/成交量 | 强度 | 次日预期 |
|---------------|------|----------|
| >30% | 强封 | 次日高开概率大 |
| 10-30% | 中等封 | 次日有分歧 |
| <10% | 弱封 | 次日低开概率大 |

### 5. 连板日的分时结构

**涨停时间窗口分析：**

| 涨停时间 | 强度 | 含义 |
|----------|------|------|
| 09:30-09:45 | 极强 | 一字板或开盘秒板，主力高度控盘 |
| 09:45-10:30 | 强 | 早盘封板，资金认可度高 |
| 10:30-13:00 | 中等 | 盘中封板，有分歧但最终封住 |
| 13:00-14:30 | 弱 | 午后封板，分歧较大 |
| 14:30-15:00 | 极弱 | 尾盘封板，次日高开低走概率极大 |

## 二、分析流程

### Step 1: 确认连板结构

- 连板天数（2板/3板/更多）
- 每日涨停时间（早盘/午后/尾盘）
- 每日封单强度
- 每日成交量与量比

### Step 2: 断板日数据采集

- 开盘价 vs 昨收（低开幅度）
- 盘中最高/最低
- 当前成交额与换手率
- 外盘/内盘比
- 分时走势（5分钟K线）

### Step 3: 板块对标分析

- 找出同板块3-5只核心股
- 获取它们当日涨跌幅
- 计算板块平均涨跌幅
- 计算背离度

### Step 4: 技术指标验证

- MACD状态（是否金叉/死叉）
- RSI超买超卖
- 均线支撑/压力
- BOLL位置

### Step 5: 关键价位判断

**压力位（从近到远）：**
1. 今日最高价
2. 昨日收盘价（涨停价）
3. 昨日开盘价
4. 前一日涨停收盘价

**支撑位（从近到远）：**
1. 今日最低价
2. 跌停价
3. 前一日涨停收盘价
4. MA20位置

## 三、情景推演

### 情景A：板块共振回调（正常调整）

**条件：** 板块同步回调，背离度 < 5%
**概率：** 30-40%
**特征：** 缩量回调，外盘/内盘比正常
**预期：** 短暂调整后可能继续上行
**操作：** 持有观察，跌破MA10考虑减仓

### 情景B：分歧加大（弱于板块）

**条件：** 板块强个股弱，背离度 5-10%
**概率：** 30-40%
**特征：** 放量下跌，内盘大于外盘
**预期：** 分歧期，方向不确定
**操作：** 减仓观望，等方向明确

### 情景C：游资撤退（严重背离）

**条件：** 板块暴涨个股暴跌，背离度 > 10%
**概率：** 20-30%
**特征：** 大幅低开，放量下跌，内盘远大于外盘
**预期：** 连续回调概率大
**操作：** 止损/清仓，不抄底

## 四、案例参考：红宝丽 002165.SZ @ 2026-06-12

### 基本信息
- 代码：002165.SZ
- 名称：红宝丽
- 行业：化工/聚氨酯
- 主营：聚醚多元醇、异丙醇胺
- 总市值：55.73亿

### 连板结构
- 6/10: +9.94% (准涨停) 量43.2万手
- 6/11: +10.00% (涨停) 量175.8万手 (量比2.04)
- 6/12: -5.60% (断板) 量98.2万手

### 板块背离分析
- 万华化学: +6.30%
- 华鲁恒升: +6.63%
- 鲁西化工: +6.97%
- 卫星化学: +5.29%
- **板块平均: +6.30%**
- **红宝丽: -5.60%**
- **背离度: 11.9% (>10%, 严重背离)**

### 断板日结构
- 开盘7.44 (低开-7.3%)
- 最高7.85 / 最低7.44
- 换手率13.54% (半天)
- 外盘/内盘: 0.84 (卖压大)
- RSI6: 72.8 (超买区)

### 判断结论
- 主力行为: **游资撤退/高位派发**
- 核心依据: 板块暴涨6%+，个股逆势跌5.6%，严重背离
- 次日预期: 继续回调概率大，关注7.30-7.44支撑
- 操作建议: 持仓者减仓，空仓者不参与
