# 交易决策推演框架 (Trade Decision Playbook)

> 2026-06-12 实战教训：用户问"如果守住缺口是否可以入场"，我只给了偏空结论，没有给出结构化的入场方案。用户期望看到多场景推演+盈亏比+仓位建议。

## 一、触发条件

当用户提出以下类型问题时，激活本框架：

- "能不能买" / "能不能入" / "能不能上"
- "如果守住XX支撑，可以入场吗"
- "什么位置可以接" / "回调到哪里可以买"
- "要不要跑" / "要不要走" / "要不要割"
- "现在该怎么操作"

**不要只给方向性判断（偏多/偏空），必须给出具体的操作方案。**

## 二、推演框架

### Step 1: 明确当前位置

先回答"现在在哪"：
- 当前价 vs 关键支撑位
- 当前价 vs 关键压力位
- 距支撑的缓冲空间（%）
- 距压力的反弹空间（%）

### Step 2: 识别支撑/压力体系

按权重从高到低排列（不可遗漏）：

| 优先级 | 支撑/压力类型 | 说明 |
|--------|-------------|------|
| 1 | 缺口支撑/压力 | 未回补缺口 > 均线，因为缺口代表主力真金白银的断层 |
| 2 | 前高/前低 | 历史关键价位（涨停价、跌停价、大阴/大阳极值） |
| 3 | 均线支撑/压力 | MA20/MA30（波段）、MA5/MA10（短线） |
| 4 | 筹码密集区 | cyq_perf 的 cost_50/cost_85 |
| 5 | 整数关口 | 心理价位（如10.00、15.00） |

### Step 3: 设计入场方案

至少提供两套方案，覆盖不同风格：

#### 方案A: 激进入场（确认性弱，成本好）

```
入场条件: 股价触及支撑位附近
入场价位: 支撑位上沿 ± 0.5%
止损位:   支撑位下沿下方 1-2%
止盈位:   分 2-3 档
仓位:     ≤2成
```

#### 方案B: 稳健入场（确认性强，成本高）

```
入场条件: 确认支撑有效后的反弹信号（放量阳线/收复关键位）
入场价位: 确认信号出现后的收盘价或次日开盘
止损位:   确认信号的最低点下方
止盈位:   分 2-3 档
仓位:     ≤3成
```

### Step 4: 盈亏比计算

**每个方案必须算盈亏比**：

```
盈亏比 = (目标价 - 入场价) / (入场价 - 止损价)

判断标准:
  ≥ 3:1 → 优秀，值得做
  ≥ 2:1 → 良好，可以做
  ≥ 1.5:1 → 勉强，谨慎做
  < 1.5:1 → 不值得，放弃
```

**盈亏比表格必须输出**：

| 方案 | 入场 | 止损 | 目标1 | 目标2 | 目标3 | 盈亏比 |
|------|------|------|-------|-------|-------|--------|
| A 激进 | X.XX | X.XX(-X.X%) | X.XX(+X.X%) | X.XX(+X.X%) | X.XX(+X.X%) | X.X:1 |
| B 稳健 | X.XX | X.XX(-X.X%) | X.XX(+X.X%) | X.XX(+X.X%) | X.XX(+X.X%) | X.X:1 |

### Step 5: 多空因素对比

**必须同时列出多头和空头因素**，不能只说一边：

#### 多头因素（支持入场）
- 资金面：主力介入证据、涨停缺口、放量建仓
- 技术面：支撑位有效、超跌反弹需求、底背离
- 消息面：新催化、板块共振
- 筹码面：缩量回调、筹码稳定

#### 空头因素（反对入场）
- 趋势面：大周期空头排列、均线压制
- 量价面：放量下跌、反弹无量
- 资金面：主力出逃、融资融券偏空
- 消息面：利空未消化、旧消息重炒

**输出格式**：

```
【多头因素】              【空头因素】
① 因素 + 权重            ① 因素 + 权重
② 因素 + 权重            ② 因素 + 权重
③ 因素 + 权重            ③ 因素 + 权重
```

### Step 6: 确认信号与失效信号

#### 入场确认信号（满足越多越可靠）

| 信号 | 权重 | 说明 |
|------|------|------|
| 缩量回踩不破支撑 | 必要 | 筹码稳定，多头防线有效 |
| 分时有明显托单 | 加分 | 有资金在支撑位承接 |
| 次日放量阳线收复 | 强确认 | 多头反攻确认 |
| 板块共振 | 加分 | 不是个股独撑 |
| 缺口形成时放量 | 加分 | 主力真金白银建仓 |

#### 失效信号（出现任一则放弃/止损）

| 信号 | 说明 |
|------|------|
| 放量跌破支撑位 | 多头防线失守 |
| 连续2日收盘低于支撑 | 趋势性破位 |
| 支撑位附近无反弹直接下杀 | 无承接 |
| 板块整体走弱 | 逆风环境 |

### Step 7: 仓位与风控

```
仓位建议:
  - 试探仓: ≤2成（支撑刚确认，不确定性高）
  - 确认仓: ≤3成（确认信号出现，加仓）
  - 总仓位上限: ≤5成（大趋势偏空时）

止盈策略:
  - 目标1到位: 减1/3
  - 目标2到位: 再减1/3
  - 目标3到位: 清仓

止损策略:
  - 跌破止损位: 全部走人，不犹豫
  - 不设"再看看"的模糊止损
```

### Step 8: 综合建议

必须给出明确的操作建议，不能只说"可以考虑"：

```
建议分类:
  ✅ 可入场: 给出具体方案（A或B）
  ⚠️ 可轻仓试探: 仓位≤2成，止损必须严格
  ⏸️ 观望等确认: 等XX信号出现再入场
  ❌ 不建议入场: 给出原因和替代策略（等回调/等突破）
```

## 四、实战案例

### 案例: 红宝丽(002165) 2026-06-12

**背景**: 6/10~6/11连拉两个涨停后,6/12跳空低开,现价7.58

**当前位置**:
- 支撑: 向上缺口7.30~7.50(涨停缺口,未回补)
- 压力: 向下缺口7.85~8.03(未回补)
- 现价7.58处于两个缺口之间的"夹心"位置

**方案A(激进)**: 7.50入场 | 止损7.25(-3.3%) | 目标7.85/8.03/8.50 | 盈亏比1.4/2.1/4.0
**方案B(稳健)**: 7.70入场(放量收复) | 止损7.45(-3.2%) | 目标8.03/8.50/9.00 | 盈亏比1.3/3.2/5.2

**多头**: ①连板主力介入 ②涨停缺口支撑 ③今日7.44探底回升有护盘
**空头**: ①周线全空头 ②今日放巨量下跌 ③上方9~10元套牢密集

**综合**: ⚠️可轻仓试探(≤3成),止损7.25,短线思维不恋战
