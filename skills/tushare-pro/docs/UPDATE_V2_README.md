# Update Daily V2 - 智能增量更新脚本

## 功能特性

### 1. 时间同步验证
- 自动获取网络时间
- 验证本地时间准确性
- 偏差超过 5 分钟发出警告

### 2. 交易日历获取
- 自动从 Tushare 获取最近一年的交易日历
- 智能识别节假日和非交易日
- 数据缓存（同一天内只获取一次）

### 3. 本地数据检测
- 自动检测每个接口的本地最新数据日期
- 支持按日期分文件的接口（如 `stk_limit_20260409.csv`）
- 支持按股票分文件的接口（如 `daily_000001.SZ.csv`）

### 4. 智能缺失补全
- 自动计算缺失的交易日
- 支持批量补全多日数据
- 默认最多补全最近 30 天（避免一次性请求过多）

## 使用方法

### 基本用法

```bash
cd ~/.openclaw/skills/custom/tushare_pro
export TUSHARE_TOKEN="68e53ce462eb8689c5e8ed6422bcd9c2be589df0c179564a4e827b9472d0"

# 智能检测并补全所有缺失数据
python3 update_daily_v2.py

# 只更新核心组（daily, daily_basic, moneyflow）
python3 update_daily_v2.py --group core

# 只更新涨停相关接口
python3 update_daily_v2.py --group limit

# 只更新单个接口
python3 update_daily_v2.py daily

# 补全最近 5 个交易日的数据（不管本地是否有）
python3 update_daily_v2.py --backfill 5

# 指定日期更新（兼容旧模式）
python3 update_daily_v2.py --date 20260409
```

### 分组说明

| 分组 | 接口 | 说明 |
|-----|------|-----|
| `core` | daily, daily_basic, moneyflow | 核心行情数据 |
| `limit` | stk_nineturn, limit_step, limit_list_d 等 | 涨停/跌停数据 |
| `moneyflow` | moneyflow_dc, moneyflow_ths, moneyflow_hsgt 等 | 资金流向数据 |
| `repurchase` | repurchase | 回购数据 |
| `block` | block_trade | 大宗交易 |
| `shock` | stk_shock, stk_high_shock | 异动数据 |
| `all` | 所有接口 | 全部更新 |

## 智能检测逻辑

1. **获取交易日历**: 从 Tushare 获取最近一年的所有交易日
2. **检测本地最新日期**: 读取本地数据文件，找出最新日期
3. **计算缺失日期**: 对比交易日历和本地日期，找出缺失的交易日
4. **批量更新**: 逐个获取缺失日期的数据

## 注意事项

1. **时间偏差**: 本地时间与网络时间偏差 8 小时是正常的（东八区），超过 5 分钟会警告但不影响执行
2. **数据无数据**: 如果某日期无数据（如非交易日或数据尚未生成），会显示 "⚪ 无数据" 但不算失败
3. **API 限流**: 脚本内置 0.2-0.3 秒延迟，避免触发 Tushare API 限流

## 与旧版脚本的区别

| 特性 | 旧版 (update_daily.py) | 新版 (update_daily_v2.py) |
|-----|----------------------|-------------------------|
| 时间验证 | ❌ 无 | ✅ 自动验证 |
| 交易日历 | ❌ 简单判断 | ✅ 完整日历 |
| 缺失检测 | ❌ 只能更新指定日期 | ✅ 自动检测缺失 |
| 批量补全 | ❌ 单次更新 | ✅ 批量补全多日期 |
| 智能跳过 | ❌ 无 | ✅ 已更新日期自动跳过 |

## 建议用法

**收盘后自动更新**:
```bash
# 添加到 crontab，每天 15:30 执行
30 15 * * 1-5 cd ~/.openclaw/skills/custom/tushare_pro && export TUSHARE_TOKEN="xxx" && python3 update_daily_v2.py --group core
```

**手动补全历史数据**:
```bash
# 补全最近 10 天的所有数据
python3 update_daily_v2.py --backfill 10
```
